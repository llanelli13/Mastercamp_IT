# Banquise 

## Project setup
### Frontend
```
cd Frontend
npm i 
npm install filepond vue-filepond
```

### Backend
```
cd Backend
npm i
```
## Starting the servers
### Frontend
```
npm run serve
```

### Backend
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
